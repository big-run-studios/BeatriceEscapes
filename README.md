# Bea Escapes

A family-friendly action roguelite beat 'em up for the web.

The Bell family must escape town together after 6-year-old Bea accidentally reveals her magical powers. Fight through procedurally assembled encounters, collect magical boons, and discover a hidden wizard community — all in 25-35 minute runs that advance the story toward a true ending.

## Tech Stack

- **Engine:** Phaser 3
- **Language:** TypeScript
- **Build:** Vite
- **Deploy:** GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Versioning

- **Design docs:** `D<major>.<minor>` — see `VERSION` file
- **Builds:** `B0.<layer>.<patch>` — tied to onion development layers

See `docs/game-pillars-and-prototype-north-star.md` for the full design document.
